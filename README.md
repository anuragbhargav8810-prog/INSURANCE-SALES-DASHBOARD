# 📊 Insurance Data Analysis Dashboard

## 🏢 Project Title

**Universal Life Insurance Data Analysis Dashboard**

---

## 📌 Overview

This project presents an interactive data visualization dashboard for analyzing insurance data. It helps in understanding customer behavior, regional trends, and the impact of factors like smoking, age, and BMI on insurance charges.

---

## 🎯 Objectives

* Analyze insurance charges across different regions
* Study the impact of smoking vs non-smoking habits
* Understand how age and BMI affect insurance costs
* Compare charges based on gender and demographics

---

## 📁 Dataset Features

The dataset includes the following attributes:

* Age
* Gender (Sex)
* BMI (Body Mass Index)
* Region (Northeast, Northwest, Southeast, Southwest)
* Smoker / Non-Smoker
* Insurance Charges

---

## 📊 Dashboard Highlights

### 🔹 Region-wise Analysis

* Southeast region shows the highest insurance charges
* Northeast and Northwest have relatively lower charges

### 🔹 Smoker vs Non-Smoker

* Smokers have significantly higher insurance costs
* Majority of customers are non-smokers

### 🔹 BMI Analysis

* Higher BMI is associated with increased insurance charges
* Southeast region shows higher average BMI

### 🔹 Age Filter

* Interactive age filter allows dynamic data exploration

### 🔹 Gender Distribution

* Data is segmented into male and female categories

---

## 📈 Visualizations Used

* Bar Charts
* Pie Charts
* Horizontal Comparison Charts
* KPI Metrics
* Interactive Filters (Age, Region, Smoker Status)

---

## 🛠️ Tools & Technologies

* Python (Pandas, Matplotlib, Seaborn)
* Power BI / Tableau (for dashboard creation)
* Data Cleaning & Visualization Techniques

---

## 🚀 Key Insights

* Smoking is the biggest factor affecting insurance charges
* Regional differences impact pricing significantly
* Age and BMI show strong correlation with insurance cost
